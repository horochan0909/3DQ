# 3DQとは
3DCGの情報を対象にしたsns風情報発信システムで利用者のインプットアウトプットに
役立つアプリケーションを目的に作成しました。
# 使用方法
非ログインユーザー
投稿された記事の閲覧、ユーザーのマイページの閲覧が可能です。
ログインユーザー
画像付きで情報を投稿しテキストだけでは伝わらない情報共有ができます。
